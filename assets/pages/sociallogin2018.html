Integrating Social Login onto an Exisiting Service.

2018, A Ticket Commerce Site (Contract/Part-time, Remote)

Introduction & Goal:

The clients wanted to add two major social logins to the existing site that already have own login and the Facebook login by SDK.

Role:

Using SDK has a benefit for implementing a specific service but it can be more complex in multiple services because each SDK has own specification. So, I designed new processes by OAuth2 specification and implemented.

Result:

The login process became simpler and had the same look and feel.
  • Front-end: Javascript, jQuery
  • Back-end: PHP5, Apache, OAuth2
